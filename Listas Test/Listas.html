Mis metas como desarrollador web

En este artículo haré una lista de mis metas a corto, mediano y largo plazo, usando listas ordenadas y no ordenadas.

Metas a corto plazo

  • Terminar de aprender HTML.
  • Dominar CSS para mejorar el diseño.
  • Crear mi primera pagina personal.

Metas a mediano plazo

  1. Aprender JavaScript.
  2. Subir mis proyectos a Github.
  3. Construir un portafolio profesional.

Metas a largo plazo

  • Trabajar como desarrollador freelancer.
    • Conseguir mis primeros clientes.
    • Ofrecer servicios de diseño web.
  • Enseñar a otros lo que aprendí.
  • Desarrollar un proyecto grande (como una app o un juego de Roblox.)
